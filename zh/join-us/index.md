# 加入我们

XShen Lab 欢迎对细胞核组织、基因组调控、转录、非编码 RNA、单细胞基因组学和细胞命运感兴趣的学生与研究人员加入。

我们的研究关注细胞核结构与功能的物理机制、基因组组织、转录、非编码 RNA 和细胞命运，综合运用分子与细胞生物学、单细胞基因组学、成像、生化方法、计算分析和理论建模。

{% include section.html %}

## 研究生

我们欢迎对分子细胞生物学、基因组学、表观遗传学、RNA 生物学、单细胞技术、计算生物学或定量生物学感兴趣的研究生。

潜在研究方向包括：

- 细胞核 RNA 与非编码 RNA 调控；
- 基因组组织与染色质结构；
- 转录调控与相分离；
- 单细胞新生转录；
- 细胞命运决定与细胞身份。

具有实验或计算背景的同学均可与我们联系。

{% include section.html %}

## 博士后

我们欢迎在以下一个或多个领域具备扎实训练的博士后研究人员：

- 分子与细胞生物学；
- 干细胞生物学与发育生物学；
- RNA 生物学与转录调控；
- 染色质生物学与细胞核组织；
- 单细胞基因组学与多组学；
- 成像、生物物理学或理论建模；
- 计算基因组学与生物信息学。

我们鼓励博士后申请者围绕实验室对基因组信息流与细胞核功能的总体兴趣提出独立科学问题。

{% include section.html %}

## 科研助理

我们欢迎有兴趣支持实验、计算或交叉学科项目的科研助理。

相关经验可包括：

- 分子克隆与细胞培养；
- 测序文库构建；
- 成像与显微技术；
- 基因组数据分析；
- 流程开发与数据管理；
- 实验室组织与项目协调。

{% include section.html %}

## 本科生

欢迎有志于接受科研训练的本科生联系实验室。

学生可根据个人兴趣与背景，参与分子生物学、基因组学、RNA 生物学、单细胞分析、计算生物学或定量建模相关项目。

{% include section.html %}

## 计算与定量生物学

我们尤其关注能够连接生物学与计算研究的学生和科研人员。

相关背景包括：

- 生物信息学；
- 计算基因组学；
- 统计学与数据科学；
- 机器学习；
- 数学建模；
- 物理学或定量生物学；
- 软件与流程开发。
